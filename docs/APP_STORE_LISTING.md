## Description

Carrier Route Pilot gives mail carriers a faster way to manage route knowledge, find stops, scan parcels, and keep delivery details organized throughout the day.

Built for real carrier workflows, it keeps route information practical, searchable, and easy to use from the street.

Key features:

- Parcel OCR scanning to match labels to route stops
- Voice address lookup for hands-free searching
- Route management with stops, addresses, notes, and mailbox types
- Public route gallery for shared route templates
- Multi-stop navigation into Apple Maps or Google Maps
- Persistent stop notes for route-specific reminders
- Curbside route maps with street segments and notable stops
- Premium route library access and AI-assisted route creation

Why carriers use it:

- Find the right stop faster
- Keep route details in one place
- Reduce repeated lookup work
- Build cleaner route references over time
- Use camera and voice tools when typing is inconvenient

Carrier Route Pilot is designed to feel simple, native, and focused. It does not replace official USPS systems or policies. It is a productivity tool for organizing route information you choose to enter and use.

Carrier Route Pilot Pro is available as a monthly subscription and unlocks premium route library access plus AI-assisted route creation tools.
